# ✈️ European Flight Traffic Analysis

> **My best project:** Cleaning, analyzing, and predicting `flt_tot` – the total daily IFR movements at European airports.

## 🎯 What is `flt_tot`?

`flt_tot` = **Total number of flights (arrivals + departures)** at an airport per day.  
It's the single most important metric for airport activity.

## 📊 What I did

| Task | Method |
|------|--------|
| Clean missing values | Pandas, interpolation |
| Detect outliers | IQR & visualization |
| Analyze trends | Time-series plots |
| *(Next)* Predict future values | LSTM / XGBoost |

## 🔍 Quick example

| date       | airport | flt_tot |
|------------|---------|---------|
| 2024-06-01 | LHR     | 1290    |
| 2024-06-01 | CDG     | 1130    |

> London Heathrow (LHR) handles ~1290 flights/day on average.

# 🏠 House Price Prediction with XGBoost

> Predict house prices using features like square footage, bedrooms, bathrooms, brick construction, and neighborhood.

## 📊 Dataset

| Feature | Type | Description |
|---------|------|-------------|
| Home | ID | Unique home identifier (dropped) |
| SqFt | numeric | Square footage |
| Bedrooms | numeric | Number of bedrooms |
| Bathrooms | numeric | Number of bathrooms |
| Offers | numeric | Number of offers received |
| Brick | categorical | Yes/No (encoded to 1/0) |
| Neighborhood | categorical | East/North/West (label encoded) |
| Price | numeric | Target variable (sale price) |

## 🚀 Model Performance

| Metric | Value |
|--------|-------|
| R² Score | `0.7918` |
| MAE | `$8,479` |
| RMSE | `$11,076` |
